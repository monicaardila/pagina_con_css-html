Principios Básicos de Calidad de Software

El Aseguramiento de la Calidad del Software es el conjunto de actividades planificadas y sistemáticas necesarias para aportar la confianza que el software satisfará los requisitos dados de calidad.

ISO 9001

Es una norma ISO internacional elaborada por la Organización Internacional para la Estandarización (ISO) que se aplica a los Sistemas de Gestión de Calidad de organizaciones públicas y privadas. independientemente de su tamaño o actividad empresarial.

IEEE 829

También conocido como el Estándar 829 para la documentación de prueba de software y sistema, era un estándar IEEE que especificaba la forma de un conjunto de documentos para su uso en ocho etapas definidas de prueba de software y prueba de Sistemas.

ISTQB

International Software Testing Qualifications Board (www.istqb.org), es una asociación sin fines de lucro (non-profit association) fundada en 2002. Su sede se encuentra en Bélgica y cuenta con constitución, normas y reglamentos propios.
